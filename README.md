# Complaint Management System

## 📌 Project Overview
The Complaint Management System is a web-based application designed to efficiently manage and resolve customer complaints.

## 🚀 Features
- Submit complaints
- View complaint list
- Track complaint status
- Simple and user-friendly interface

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript
- SQL

## 📂 Project Structure
complaint-management-system/
│── index.html
│── style.css
│── script.js
│── README.md
└── database/
└── schema.sql

## ⚙️ How to Run
1. Download or clone the repository
2. Open `index.html` in a browser
3. Use the form to submit complaints

## 📈 Future Improvements
